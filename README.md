### One-click setup: run R in your browser

1. Create a free account on [GitHub](https://github.com)
2. Enable [GitHub Education](https://github.com/education), which gives you access to many developer tools for free, including running R in your browser and using AI tools.
3. Click the green "Code" button above and select "Open with Codespaces".
